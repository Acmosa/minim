get_file_urls(quality: Union[int, str] = 27) → None[source]#

Get the file URLs for all tracks.

User authentication

Requires user authentication and an active Qobuz streaming plan to access the URL for the full, high-quality audio track.

Parameters:
qualityint or str, default: 27

Maximum audio quality.

Valid values:

  • 5 or "MP3" for constant bitrate (320 kbps) MP3.

  • 6 or "CD" for CD-quality (16-bit, 44.1 kHz) FLAC.

  • 7 for up to 24-bit, 96 kHz Hi-Res FLAC.

  • 27 or "HI-RES" for up to 24-bit, 192 kHz Hi-Res FLAC.

get_streams(*, quality: Union[int, str] = 27, save: bool = False, path: str = None, folder: bool = False, metadata: bool = True) → None[source]#

Get audio stream data for all tracks.

User authentication

Requires user authentication and an active Qobuz streaming plan for full, high-quality audio streams.

Parameters:
qualitystr, keyword-only, default: 27

Maximum audio quality.

Valid values:

  • 5 or "MP3" for constant bitrate (320 kbps) MP3.

  • 6 or "CD" for CD-quality (16-bit, 44.1 kHz) FLAC.

  • 7 for up to 24-bit, 96 kHz Hi-Res FLAC.

  • 27 or "HI-RES" for up to 24-bit, 192 kHz Hi-Res FLAC.

savebool, keyword-only, default: False

Determines whether the streams are saved to audio files.

pathstr, keyword-only, optional

If save=True, path in which the audio files are saved.

folderbool, keyword-only, default: False

Determines whether a folder in path (or in the current directory if path is not specified) is created to hold the audio files.

metadatabool, keyword-only, default: True

Determines whether the audio files’ metadata is populated.
